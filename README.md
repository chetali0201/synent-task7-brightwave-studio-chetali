# BrightWave Studio - Multi-Page Website

A responsive multi-page website developed as part of the Synent Technologies Web Development Internship.

## 📌 Project Overview

BrightWave Studio is a modern digital agency website designed using HTML, CSS, and JavaScript. The project includes multiple pages with a consistent design, responsive layout, and user-friendly navigation.

## ✨ Features

* Responsive navigation bar
* Home, About, Services, and Contact pages
* Mobile-friendly design
* Clean and modern user interface
* Contact form with basic validation
* Consistent styling across all pages

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
